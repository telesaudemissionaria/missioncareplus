Política de Privacidade

MissionCare Plus — Telesaúde Missionária

Última atualização: Março de 2026 · Versão 1.0

Este aplicativo foi desenvolvido com privacidade por design. Coletamos o mínimo necessário para funcionar e nunca armazenamos dados clínicos identificáveis.

1. Responsável pelo Aplicativo

Organização: Telesaúde Missionária

Este aplicativo é uma ferramenta educativa de apoio a leigos em situações de emergência. Não é um serviço médico.

2. Dados que Coletamos (Mínimos)

  • Preferência de idioma — armazenada localmente no seu dispositivo (localStorage)
  • Aceite dos termos de uso — registrado localmente, sem envio a servidores
  • Protocolo de emergência acessado — apenas para funcionalidade offline, sem identificação do usuário

3. Dados que NÃO Coletamos

  • Histórico de saúde ou sintomas
  • Diagnósticos ou condições médicas
  • Nome, CPF, data de nascimento ou qualquer dado pessoal identificável
  • Localização geográfica
  • Dados biométricos
  • Contatos do dispositivo

O app não exige cadastro. Você pode usar todas as funcionalidades sem criar conta.

4. Uso de Inteligência Artificial (OpenAI)

Quando você usa o assistente de IA (Dr. IA), as informações inseridas são enviadas à API da OpenAI para processar a resposta.

  • Não enviamos nome, CPF ou dados identificáveis à OpenAI
  • As consultas são tratadas como anônimas
  • A OpenAI possui sua própria política de privacidade disponível em openai.com/policies/privacy-policy
  • Quando offline, nenhum dado é enviado — o app usa protocolos locais

5. Armazenamento e Segurança

  • Preferências são armazenadas somente no seu dispositivo (localStorage/sessionStorage)
  • Não mantemos banco de dados de usuários
  • Toda comunicação com servidores externos usa HTTPS (criptografia em trânsito)
  • Não compartilhamos dados com terceiros além da OpenAI para o assistente de IA

6. Seus Direitos (LGPD — Lei 13.709/2018)

Como coletamos dados mínimos e anônimos, a maioria dos direitos da LGPD não é aplicável neste contexto. Porém, você tem direito a:

  • Apagar seus dados locais a qualquer momento (limpar dados do navegador/app)
  • Negar o uso do assistente de IA e usar apenas os protocolos offline
  • Solicitar informações sobre o tratamento de dados pelo contato abaixo

7. Cookies e Rastreamento

Este aplicativo não utiliza cookies de rastreamento, publicidade ou análise de comportamento. Usamos apenas localStorage para preferências técnicas do usuário.

8. Uso por Menores de Idade

Este aplicativo pode ser usado por adultos responsáveis em situações de emergência pediátrica. Não é direcionado a menores de 18 anos como usuários primários. Não coletamos dados de crianças.

10. Atualizações desta Política

Esta política pode ser atualizada a cada 12 meses ou quando houver mudanças significativas no app. A data da última atualização sempre estará no cabeçalho desta página.
